# Travel Group PM

## Objective

Validate whether group travel planning suffers from low participation and information fragmentation.

## Current phase

Discovery & User Research

## Hypothesis

In many group trips, most of the planning work falls on one person, creating coordination problems and reducing participation from the rest of the group.

## Deliverables

* User interviews

* Problem validation report

* Stakeholder analysis

* MVP definition

* Roadmap

🟡 Research in progress
